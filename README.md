Описание
========

Oracle PL/SQL функция, вычисляющая дату следующего запуска. На входе - дата отсчёта, матрица планировщика; на выходе - следующая дата запуска.


## Замечания к реализации

1. Вычисление дней недели зависит от системного календаря. Следовательно, это нужно учитывать при формировании матрицы.

2. В функции реализована защита от инъекций, но нет проверки корректности входных данных матрицы запусков. Можно добавить при необходимости.

3. Чтобы пройти контрольный тест, пришлось поменять логику обработки "дней недели". В каноническом варианте дни недели объединяются с днями месяца через логическое ИЛИ. В текущей реализации объединение через логическое И.
